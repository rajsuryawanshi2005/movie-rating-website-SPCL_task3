# 🎬 MovieFlix

MovieFlix is a modern movie discovery web application that allows users to search and explore movies and web series from different categories. The platform provides an interactive and user-friendly interface where users can browse Bollywood movies, Tollywood movies, and popular Indian web series.

The application integrates the OMDb API to fetch real-time movie data such as movie titles, posters, release year, genre, actors, and plot information. It also provides live search suggestions, pagination for browsing results, random movie recommendations, and a trailer feature that redirects users to YouTube.

The project is built using HTML, CSS, and JavaScript and demonstrates practical implementation of frontend development and API integration.

# 📌 Features

- 🔍 Movie search functionality  
- 💡 Live search suggestions while typing  
- 🎥 Movie details popup with full information  
- ▶️ Watch movie trailers on YouTube  
- 🎬 Bollywood movies section  
- 🎬 Tollywood movies section  
- 📺 Popular web series section  
- 🎲 Random movies section for discovering new content  
- 📄 Pagination with Previous, Next, and Page Numbers  
- ⭐ Star rating interface  
- 🎨 Modern Netflix-style user interface  
- 📱 Responsive design for different screen sizes  

# 🛠 Technologies Used

- **HTML5** – Structure of the website  
- **CSS3** – Styling and layout design  
- **JavaScript (ES6)** – Interactivity and dynamic features  
- **OMDb API** – Fetching movie data  
- **YouTube Search Integration** – Watching movie trailers  

# 🚀 Installation / How to Run

1. Clone the repository
   git clone https://github.com/yourusername/movieflex.git

2. Open the project folder
   cd movieflex

3. Open the `index.html` file in your browser.
   No additional installation is required since this is a frontend project.
